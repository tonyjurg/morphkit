# Changelog

All notable changes to Morphkit will be documented in this file.

## 1.0.0 - 2026-03-23

- Freeze the current software functionality as the 1.0.0 baseline release.
- Add standard Python packaging metadata so Morphkit can be built as a wheel and source distribution.
- Introduce single-source version metadata shared by code, packaging, and documentation.
- Prepare GitHub Actions workflows for package releases and versioned documentation publishing.
- Add versioned documentation support with `stable`, `dev`, and tagged release paths plus a docs version selector.
