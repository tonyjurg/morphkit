Releases
========

Morphkit documentation is published per software version.

- ``stable`` points to the latest released version.
- ``dev`` tracks the current ``main`` branch.
- Tagged releases are published at paths such as ``v1.0.0``.

Baseline Release
----------------

The current software baseline is frozen as ``1.0.0``. This is the first version intended to be built and released as an installable package, with matching documentation and release notes.

Using Versioned Docs
--------------------

Use the version selector in the docs sidebar to switch between:

- the latest stable release,
- the current development branch,
- older tagged releases when they are available.
